## Descarga de los apuntes

Para descargar los apuntes ejecute el siguiente comando:
```bash
git clone https://github.com/PythonistaMX/py101.git
```

## Entorno de desarrollo (Dev Container)

Este repositorio incluye una configuración de **Dev Container** para facilitar el entorno de desarrollo. 
Para utilizarlo, asegúrate de tener instalados [Docker](https://www.docker.com/) y la extensión [Dev Containers](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers) en Visual Studio Code.

Una vez abiertos los archivos en VS Code, puedes reabrir el proyecto en el contenedor haciendo clic en el botón azul de la esquina inferior izquierda o ejecutando el comando:
`Dev Containers: Reopen in Container`.

Esto preparará un entorno con Python 3 y las extensiones necesarias (Jupyter, Pylance) listas para ejecutarse.
